# Student Performance Prediction

A machine learning project that predicts students' exam scores using academic and personal factors.

## Features

- Data preprocessing
- Missing value handling
- One-Hot Encoding
- Random Forest Regression
- Feature Importance Visualization
- Model Saving

## Technologies

- Python
- Pandas
- Matplotlib
- Scikit-learn
- Joblib

## Run

```bash
pip install -r requirements.txt
python train.py
```

## Dataset Columns

- Study Hours
- Attendance
- Previous Grade
- Sleep Hours
- Gender
- Parent Education
- Internet Access
- Family Support
- Extracurricular Activities

## Output

- Model Evaluation Metrics
- Feature Importance Graph
- Saved Model (.pkl)
